# Simulasi Reservoir

Ini adalah software reservoir yang akan digunakan untuk tugas/project akhir mas zuher.

## Struktur Ringkas

- `README.md` tetap di root sebagai pintu masuk repository.
- Folder `docs/` menyimpan seluruh dokumen planning dan referensi project.
- Folder `.github/` disiapkan untuk kebutuhan GitHub repository.
- File `.gitattributes` tetap di root karena Git membacanya dari level repository.
